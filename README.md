# 🎯 Summer Internship - C++ Console Projects

A collection of beginner-friendly C++ console applications developed as part of my Summer Internship. These projects demonstrate fundamental programming concepts including loops, conditionals, functions, arrays, user input validation, and basic game logic.

---

## 📚 Projects Included

### 🎲 Task 1 - Number Guessing Game

A console-based game where the computer randomly generates a number between **1 and 100**, and the player attempts to guess it.

#### Features
- Random number generation
- User input handling
- Higher/Lower hints
- Counts total attempts
- Simple and interactive gameplay

**Concepts Used**
- `rand()`
- `srand()`
- Loops
- Conditional statements
- User input/output

---

### 🧮 Task 2 - Simple Calculator

A basic calculator capable of performing arithmetic operations on two numbers.

#### Supported Operations
- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division

#### Features
- Operator selection
- Division by zero handling
- Invalid operator detection
- Easy-to-use console interface

**Concepts Used**
- Switch statements
- Arithmetic operators
- Conditional logic
- Input validation

---

### ❌⭕ Task 3 - Tic-Tac-Toe Game

A two-player console implementation of the classic Tic-Tac-Toe game.

#### Features
- 3×3 game board
- Two-player gameplay
- Input validation
- Win detection
- Draw detection
- Play again option

**Concepts Used**
- Functions
- Arrays (`std::array`)
- Loops
- Input validation
- Game logic
- Modular programming
